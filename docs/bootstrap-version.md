# ZERO UI Bootstrap

O pacote `@zero-ui/bootstrap` reaproveita Bootstrap 5.3+ e `react-bootstrap`, adicionando tokens, variantes e estilos proprios com prefixo `zui-*`.

Importacao recomendada:

```tsx
import "@zero-ui/bootstrap/styles.css";
import { Button, Card, Input } from "@zero-ui/bootstrap";
```

Dependencias esperadas:

```bash
pnpm add bootstrap react-bootstrap
```
