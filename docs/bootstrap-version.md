# ZERO UI Bootstrap

O pacote `zeroui-bootstrap` reaproveita Bootstrap 5.3+ e `react-bootstrap`, adicionando tokens, variantes e estilos proprios com prefixo `zui-*`.

Importacao recomendada:

```tsx
import "zeroui-bootstrap/styles.css";
import { Button, Card, Input } from "zeroui-bootstrap";
```

Dependencias esperadas:

```bash
pnpm add bootstrap react-bootstrap
```
