# ZERO UI design system

ZERO UI usa tokens compartilhados no pacote `@zero-ui/core` e implementacoes visuais separadas em Bootstrap e Tailwind.

## Principios

- Minimalismo premium.
- Glass controlado, sempre preservando contraste.
- Tema claro, escuro e cosmic.
- Estados visiveis de hover, focus, active, disabled e loading.
- API parecida entre Bootstrap e Tailwind.

## Prefixos

- Variaveis CSS: `--zui-*`
- Classes proprias: `zui-*`
- Aliases legados internos: `zf-*`, mantidos apenas por compatibilidade inicial.
